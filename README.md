# PELICULAS Formulario

#### Pasos para iniciar proyecto


- clonar repo
```bash
    git clone https://github.com/BrandonPaez23/view-peliculas
```

- Ingresar a carpeta
```
    cd view-peliculas
```

- instalar dependencias
```bash
    npm install
```
- iniciar servidor
```bash
    npm run dev
```
